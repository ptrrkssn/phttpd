System Requirements

In order to run Phttpd you need to have the following:
  • A Sun SPARCstation or a PC running the SunSoft Solaris 2.5 operating system (or later).
  • Atleast 5 MB of free disk space.
  • 32MB of RAM (or more) is recommended (not that Phttpd needs that much but the operating system in itself will be very slow otherwise).
  • Phttpd will need about 2.7 MB of RAM, regardless of the numer of virtual servers running.
If you are using the source distribution of Phttpd then you also need to have a working C compiler on your system (please note that by default Solaris 2.5 does not contain such a compiler). Phttpd has been compiled and tested with the following compilers:
  • Gnu C (GCC) 2.7.2
  • SunSoft SPARCompiler/SparcWorks 3.0.1
  • Apogee C 3.0
In general you should install the latestet recommended patches from SUN in case you do have problems.
Solaris 2.5 and 2.5.1 may really need that (due to some bugs in the thread system). Even if you do not have
a support contract with SUN, you can always retrieve those patches via anonymous ftp.
